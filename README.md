Like attrs, but for dicts

What it can do:

+ Generate TypedDict from pydantic models, attrs schemas, and dataclasses.
+ Validate dicts using TypedDict at runtime.

Why it's cool:

+ Supports multiple schema formats.
+ 100% type safe.

## Installation

```bash
python3 -m pip install typed-dict
```
